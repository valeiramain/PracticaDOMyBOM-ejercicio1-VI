Comenzar el juego

Instrucciones:

  1. El juego consiste en presionar el botón "Comenzar" para generar un número mágico.
  2. Ingrese el número mágico que considere que será generado por el sistema y luego presione el botón "Enviar"
  3. Si el usuario NO lo adivinó, el sistema le avizará por pantalla si el número ingresado es mayor o menor al número mágico.
  4. Cuando el usuario adivine el número mágico, el sistema le avizará por pantalla.

    Felicitaciones!!! Adivinaste el número mágico

    El número ingresado es mayor o menor que el número mágico
